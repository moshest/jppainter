We got the inspiration to create this project from the works of Jackson Pollock. This project illustrates that we can create art by using a given data set and some randomization.

Implementation

Strokes and Points

We create the painting by implementing the Strokes objects. Each Stroke has a color, width and number of control points. The purpose of the control points are to ensure that the stroke will pass through them during the painting process.

In addition to the complete stroke width, each point on the stroke had an independent width of its own. The final stroke width changes according to the distance between each two point.

The Drawing Process

The drawing process is quite simple.

The first step is to create the stroke path on the canvas, beginning with two points that are the source and target points of the stroke. We the randomly select a few more points and use the BÃ©zier curves technique with these points as control points. We improve the equation by adding the points width as a third dimension. This way we got a smooth line with nonpermanent width.

The second step is moving through this path and saving each point on it. During the saving process we choose several points (randomly) and assign them an exponential random width, in order to create the effect of paint "splashes". We were careful not to overdo, and if the change was too significant we performed balancing with the neighboring points.

The third and last part was painting the stroke on the canvas by our path points. During the painting we splash some color dots on the canvas randomly. Sometimes near the path and sometimes faraway.

The Data Files

The input to the program is two tables representing a directed graph with a weight on each arc. From this graph we create a stroke for each arc and the weight of the arc is represented by the color of the stroke. We insert some middle control points for each stroke to make it more interesting.
