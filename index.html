JP-Painter

Jackson Pollock paint simulation

View on GitHub Download .zip Download .tar.gz

We got the inspiration to create this project from the works of Jackson Pollock. This project illustrates that we can create art by using a given data set and some randomization.

Here are some videos that have helped us in understanding and developing the algorithm:

Foreign Labor Data

Foreign labor migration between countries: each paint streak represents the migration of workers between the two countries at the edges. The color of the streak is linked to magnitude - darker means larger amounts of people.
